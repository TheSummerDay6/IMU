# IMU V1.1 Pro 移植方法

## 一、准备工作

### 1.1 工程环境准备
- 确保已安装 Keil MDK-ARM 开发环境
- 准备 STM32CubeMX 用于硬件配置
- 确认目标平台为 STM32F407xx 系列

### 1.2 创建工程结构
在移植前，建议创建以下目录结构：
```
YourProject/
├── Drivers/
├── BSP/
├── Components/
│   ├── Algorithm/
│   ├── Controller/
│   ├── Devices/
│   ├── kalman_filter/
│   └── user_lib/
├── Application/
└── Src/
```

## 二、硬件配置（STM32CubeMX）

### 2.1 GPIO 配置

| GPIO | 模式 | 上拉/下拉 | 输出速度 | 功能说明 |
|------|------|-----------|----------|----------|
| PA4 | Output Push Pull | Pull-up | High | CS1_ACCEL (BMI088加速度计片选) |
| PB0 | Output Push Pull | Pull-up | High | CS1_GYRO (BMI088陀螺仪片选) |
| PC4 | External Interrupt Mode | Pull-up | - | INT1_ACCEL (加速度计中断) |
| PC5 | External Interrupt Mode | Pull-up | - | INT1_GYRO (陀螺仪中断) |


| PG3 | External Interrupt Mode | Pull-up | - | IST8310_DRDY (磁力计数据就绪，可选) |
| PG6 | Output Push Pull | Pull-up | Medium | IST8310_RST (磁力计复位，可选) |

### 2.2 SPI1 配置（BMI088通信）
- **模式**: Full-Duplex Master
- **Prescaler**: 256
- **CPOL**: High
- **CPHA**: 2 Edge
- **引脚分配**:
  - PA5: SPI1_SCK
  - PA6: SPI1_MISO
  - PA7: SPI1_MOSI (引脚记得改成PA7)

### 2.3 I2C 配置 --- 可跳过
| 接口 | 用途 | 模式 | 说明 |
|------|------|------|------|
| I2C2 | OLED显示 | 快速模式 (400kHz) | 必需 |
| I2C3 | IST8310磁力计 | 快速模式 (400kHz) | 可选 |

### 2.4 USART6 配置 --- 默认即可Asynchronous
- **波特率**: 115200
- **字长**: 8 Bits
- **停止位**: 1
- **校验位**: None
- **模式**: Asynchronous
- **中断**: 开启接收中断

### 2.5 TIM10 配置（PWM输出）
- **功能**: PWM Generation CH1
- **引脚**: PF6（记得改成PF6）
- **预分频器 (PSC)**: 0
- **自动重载值 (ARR)**: 65535

### 2.6 DMA 配置
| 类型 | 通道 | 方向 | 优先级 |
|------|------|------|--------|
| SPI1_RX | DMA2 Stream2 | Peripheral To Memory | Very High |
| SPI1_TX | DMA2 Stream3 | Memory To Peripheral | High |

### 2.7 NVIC 中断优先级配置
| 中断源 | 优先级 | 说明 |
|--------|--------|------|
| System tick timer | 15 | 系统滴答定时器 |
| EXTI Line4 interrupt | 5 | 加速度计中断 |
| EXTI Line3[9:5] interrupts | 5 | 陀螺仪中断 |
| EXTI Line3 interrupt | 5 | 磁力计中断（可选） |
| DMA2 Stream2/Stream3 | 5 | SPI DMA中断 |

## 三、文件复制

### 3.1 CMSIS-DSP 库文件
**位置**: `./Drivers/CMSIS/Lib/`
**操作**: 将 `arm_cortexM4lf_math.lib` 和 `arm_math.h` 复制到工程的 CMSIS 文件夹中

### 3.2 BSP 文件
**操作**: 将所有 BSP 相关文件复制到工程的 `BSP/` 目录中

### 3.3 Components 文件
1. `Algorithm/` - 算法模块
2. `Controller/` - 控制器模块
3. `Devices/` - BMI088驱动
4. `kalman_filter/` - 卡尔曼滤波器
5. `user_lib/` - 用户库函数

### 3.4 HAL 相关文件
从原工程复制以下文件到对应位置：
1. `Src/` 目录中的：
   - `hal_msp.c`
   - `hal_timebase_tim.c`
   - `stm32f4xx_it.c`
2. `Application/` 目录全部文件

### 3.5 启动文件
**操作**: 将 `MDK-ARM/startup_stm32f407xx.s` 文件复制到工程根目录
或者把29行和33行的 Stack_Size EQU 0x400 改成0x2000 Heap_Size EQU 0x200 改成0x2000

## 四、Keil MDK-ARM 配置

### 4.1 添加文件到工程
1. 在项目管理器中添加所有复制的文件
2. 添加头文件路径：
   - `./Drivers/CMSIS/Include`
   - `./BSP/`
   - `./Components/`
   - `./Application/`

### 4.2 Target 配置
打开 **Options for Target** → **Target** 标签：
1. 勾选 **Use MicroLIB**
2. 内存配置：
   - IRAM1: `0x20000000` 大小 `0x20000`
   - IRAM2: `0x10000000` 大小 `0x10000`

### 4.3 C/C++ 配置
打开 **Options for Target** → **C/C++** 标签：
1. 在 **Define** 中添加以下预处理器定义：
   ```
   USE_HAL_DRIVER
   STM32F407xx
   ARM_MATH_CM4
   __FPU_USED=1U
   __FPU_PRESENT=1U
   __CC_ARM
   ARM_MATH_MATRIX_CHECK
   ARM_MATH_ROUNDING
   ```
USE_HAL_DRIVER,STM32F407xx,USE_HAL_DRIVER,STM32F407xx,ARM_MATH_CM4,__FPU_USED=1U,__FPU_PRESENT=1U,__CC_ARM,ARM_MATH_MATRIX_CHECK,ARM_MATH_ROUNDING

### 4.4 Linker 配置
1. 添加 CMSIS-DSP 库路径
2. 确保链接器包含 `arm_cortexM4lf_math.lib`
